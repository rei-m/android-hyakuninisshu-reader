百人一首
========

## Description
- [https://github.com/rei-m/android_hyakuninisshu](百人一首 簡単に暗記) の読み上げ機能版。欲しい欲しい言われ続けたので作成。
- 基本的な構成は元アプリと同じ形。

## Build
- Cloneして自分の`google-services.json` を追加するか、Firebase関連のライブラリをすべて削除してください。
- DebugFlavorの読み上げデータはすべて序歌になっています。
